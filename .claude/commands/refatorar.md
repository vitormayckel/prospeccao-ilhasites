---
name: refatorar
description: Reimplementa código existente de forma mais limpa.
argument-hint: [componente ou arquivo pra refatorar]
---

O usuário vai indicar qual componente ou arquivo quer melhorar.

1. Leia o arquivo atual
2. Identifique os problemas (código duplicado, lógica confusa, performance ruim)
3. Diga: "Vou melhorar [arquivo]. Os problemas que encontrei: [lista curta]. Posso refatorar?"
4. Espere aprovação
5. Reimplemente mantendo a mesma funcionalidade
6. Diga o que mudou em 2-3 linhas
7. "Testa no navegador. Tudo deve funcionar igual, só o código tá melhor."

Sabendo tudo que você sabe agora sobre esse projeto, implemente a solução mais limpa e elegante.
