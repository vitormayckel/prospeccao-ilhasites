# Stack — por que cada tecnologia

## Next.js 14 (App Router)
Framework React com Server Components, rotas baseadas em arquivo, API routes embutidas. Não precisa de backend separado. Tudo num lugar só.

## TypeScript
JavaScript com tipos. O Claude Code erra menos quando tem tipos. Você erra menos quando tem tipos. Todo mundo ganha.

## Tailwind CSS
Estilização direto no HTML. Não precisa criar arquivos CSS separados. O Claude Code entende perfeitamente.

## Shadcn/ui
Componentes prontos (botão, input, modal, select, etc) que parecem profissionais. Você não precisa desenhar UI do zero.

## Supabase
Banco de dados + autenticação + storage de arquivos. Tudo numa plataforma gratuita pra começar. Interface visual pra criar tabelas sem SQL.

## Framer Motion
Animações suaves. Fade-in, slide, hover effects. Faz o projeto parecer mais polido com pouco esforço.

## Lucide React
Ícones SVG leves e bonitos. Milhares de opções. Mesma linguagem visual em todo o projeto.

## Vercel
Deploy em 2 cliques. Importa do GitHub, configura variáveis, clica deploy. URL funcionando em 2 minutos.

## Por que essa stack?

1. Tudo funciona junto sem configuração complexa
2. O Claude Code conhece tudo isso profundamente
3. Deploy gratuito pra projetos pequenos/médios
4. O aluno da imersão consegue usar sem saber programar
5. Projetos profissionais usam essa stack (não é brinquedo)
6. Escala quando o projeto crescer
